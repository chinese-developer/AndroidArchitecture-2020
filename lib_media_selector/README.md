# 多媒体文件选择库

    目前基于boxing修改

## 1 AndroidN  在 FileProvider 的 xm l配置中加入：

```        
    <cache-path name="internal" path="boxing" />
    <external-path name="external" path="DCIM/bili/boxing" />
```

## 2 So库

```
                defaultConfig {
                       ......
                      ndk {// 只打包armeabi架构的so库
                          abiFilters 'armeabi'
                      }
                }
```
